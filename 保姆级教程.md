## 🏗️ 第一步：准备工作 (只需做一次)

在开始之前，我们需要给电脑装两个基础软件：**Python** (后端大脑) 和 **Node.js** (前端界面)。

### 💻 对于 Windows 用户

1.  **安装 Python**:
    *   👉 [点击这里下载 Python 3.10+](https://www.python.org/downloads/)
    *   🛑 **关键一步**：安装界面下方有一个 **"Add Python to PATH"** (添加到环境变量) 的勾选框，**一定要勾上！一定要勾上！一定要勾上！** (重要的事情说三遍)
    *   然后一路点击 "Install Now" 即可。

2.  **安装 Node.js**:
    *   👉 [点击这里下载 Node.js (LTS 版本)](https://nodejs.org/)
    *   下载后双击安装包，一路点击 "Next" (下一步) 直到完成。

### 🍎 对于 macOS (苹果) 用户

1.  打开你的终端 (按 `Command + 空格`，输入 `Terminal` 回车)。
2.  如果你安装了 Homebrew，直接运行：
    ```bash
    brew install python node
    ```
3.  如果没有，请去官网下载安装包（链接同上），像安装普通软件一样安装即可。

## 🚀 第二步：下载与加速 (让网络飞起来)

为了防止下载依赖时卡住，我们需要配置一下国内的高速镜像源。

### 1. 下载项目代码
如果你拿到了这个文件夹，请直接解压到桌面。
在文件夹内，**右键**空白处 -> 选择 **"在终端中打开"** (Windows 11) 或 **"Git Bash Here"**。

### 2. 配置加速源 (在终端里复制粘贴并回车)

**Windows (Powershell) / macOS**:
```bash
# 设置 Python 镜像源为清华大学源
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# 设置 Node.js 镜像源为淘宝源
npm config set registry https://registry.npmmirror.com
```

## 🛠️ 第三步：一键安装依赖

我们需要分别给后端和前端安装“零件”。请打开两个终端窗口（或者分两步走）。

### 1. 安装后端 (Backend) 依赖
在终端中依次输入以下命令（每行输入完按回车）：

```bash
cd backend
# 创建一个虚拟环境（相当于给 Python 穿个独立马甲，互不影响）
python -m venv venv

# 激活虚拟环境
# Windows 用户输入:
.\venv\Scripts\activate
# macOS 用户输入:
source venv/bin/activate

# 安装所有需要的零件
pip install -r requirements.txt
```
✅ **成功标志**：最后一行显示 `Successfully installed...`，没有红色的 Error。

### 2. 配置 API Key (关键！)
1.  进入 `backend` 文件夹。
2.  找到 `.env.example` 文件，复制一份，重命名为 `.env`。
3.  用记事本打开 `.env`，把 `DEEPSEEK_API_KEY=sk-xxx` 这一行改成你真实的 Key。

### 3. 安装前端 (Frontend) 依赖
新开一个终端窗口（或者退回上一级目录），进入 `frontend` 文件夹：

```bash
cd frontend
npm install
```
✅ **成功标志**：显示 `added X packages`，可能会有一些黄色的 warning，只要不是红色 Error 就不用管。

## ▶️ 第四步：启动程序！

激动人心的时刻到了！我们需要启动两个服务。

### 1. 启动后端 (大脑)
在 `backend` 目录的终端里（确保前面有 `(venv)` 字样）：
```bash
# Windows
.\venv\Scripts\uvicorn main:app --host 0.0.0.0 --port 8000

# macOS
./venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000
```
🎉 **看到这个就成功了**：
`INFO: Uvicorn running on http://0.0.0.0:8000`

### 2. 启动前端 (界面)
在 `frontend` 目录的终端里：
```bash
npm run dev
```
🎉 **看到这个就成功了**：
`➜ Local: http://localhost:5173/`

## ❓ 常见问题急救包 (Troubleshooting)

### Q1: 运行 pip install 时出现满屏红字？
*   **原因**: 可能是网络超时，或者 Windows 缺少 C++ 编译环境。
*   **解决**: 
    1. 确认你已经执行了“第二步”的镜像加速命令。
    2. 尝试运行 `pip install --upgrade pip` 升级一下工具。

### Q2: 启动后端时提示 "uvicorn 不是内部或外部命令"？
*   **原因**: 虚拟环境没激活，或者没安装进去。
*   **解决**: 
    1. 确保你先运行了 `activate` 命令（终端前面有 `(venv)` 括号）。
    2. 尝试直接用完整路径运行：`python -m uvicorn main:app ...`

### Q3: 网页能打开，但点击“开始审计”报错 "Authentication Fails"？
*   **原因**: 你的 API Key 没填对，或者没保存。
*   **解决**: 
    1. 检查 `backend/.env` 文件，确保 `DEEPSEEK_API_KEY` 后面没有多余的空格。
    2. 修改文件后，**必须重启后端终端**（按 Ctrl+C 停止，再重新输入启动命令）才会生效！
